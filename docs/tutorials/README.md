# CEA Tutorials

Discoverability hub for everything under `docs/tutorials/`. The Sphinx-built developer reference (RST) lives separately; everything below is Markdown that renders directly on GitHub or in any Markdown viewer.

## For users

- **[CEA-4 App Training Guide](cea4-app-training/index.md)** — Feature-by-feature walkthrough of the dashboard (41 features across 10 categories).
- **[Quick Reference](cea4-app-training/00-quick-reference.md)** — Single-page overview of every dashboard feature.
- **[Variable & File Glossary](glossary/index.md)** — All 179 input / intermediate / output files, grouped by feature, generated from `cea/schemas.yml`.

## Maintaining this directory

- **`cea4-app-training/`** — hand-written, edited directly.
- **`glossary/`** — generated by `scripts/generate_tutorial_glossary.py`; do not hand-edit.
- **`how-tos/`** — hand-written developer how-tos; edited directly.
- **`index.rst`** — Sphinx index; references the RST-only developer guides and links to this Markdown tree.
