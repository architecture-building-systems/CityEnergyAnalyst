How to use the T2 tool
----------------------

######### Packages to install
- deap (for the evolutionary algorithm)


######### Folder structure for data saving

The following subfolders have to be present :
- Raw        :	 Raw data from J+
- SubsRes    :   Substation results for disconnected buildings
- ClustRes   :   Clustering results for disconnected buildings
- DiscRes    :   Operation pattern for disconnected buildings
- TotalNtw   :   Total files (inputs to substation + ntw in master)
- NtwRes     :   Ntw summary results
- MasterRes  :   Master checkpoints
- SolarRaw   :   Raw solar files
- SlaveRes   :   Slave results (storage + operation pattern)
- NtwLayout  :   Ntw layout files


########## How to run the tool

- from UESM_Main
- inputs to change in globalVar
