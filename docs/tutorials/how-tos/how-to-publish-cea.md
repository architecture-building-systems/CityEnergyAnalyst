# How to publish CEA to Zenodo

This guide covers the main steps of publishing City Energy Analyst to [Zenodo](https://zenodo.org/).

1. Create a release on GitHub. Zenodo automatically creates an immutable copy of the CEA repository and issues a DOI.
2. Log in to [Zenodo](https://zenodo.org/). The username and password are stored in the `cea@arch.ethz.ch` account.
3. Go to **Home → Account → GitHub**.
4. Click on the `architecture-building-systems/CityEnergyAnalyst` repository.
5. Click on the DOI of the latest released version. This opens the citation editor.
6. Click **Edit**.
7. Update the author list and order based on the number of additions merged into `master` at the date of release.
8. Add the current PI as the last author.
9. Click **Save changes** and **Publish**.
