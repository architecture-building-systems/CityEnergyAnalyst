Installation guide for macOS
==============================

.. include:: ../common/cea-interfaces.rst

Step 1: Download and install CEA Desktop App
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
#. Download the latest version of CEA `here <https://www.cityenergyanalyst.com/#downloads>`__. (we support both Intel and Apple Silicon)
#. Open the .dmg file and drag the .app into your Applications folder.

Step 2: Run CEA Desktop App
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

#. Locate the CEA application in the Applications folder or Launchpad and double click on it
#. Start using CEA Desktop App
